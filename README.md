# NutriTrack – Premium Health & Nutrition Dashboard

NutriTrack is a state-of-the-art health tracking ecosystem designed to optimize metabolic health through intelligent meal logging, AI-driven behavioral analysis, and milestone-based motivation. Built with a premium "Healthcare SaaS" aesthetic, it offers a seamless experience across desktop and mobile devices.

---

## 🚀 Key Features

### 🧠 Intelligent Dashboard & Analytics
- **Dynamic Health Scoring**: Real-time calculation of your weekly health percentage based on dietary quality.
- **Behavioral Maps**: High-density performance charts visualizing healthy vs. unhealthy meal distributions.
- **Neural Insights (AI)**: Advanced pattern detection that identifies high-risk days and suggests metabolic optimizations.

### 🍱 Strategic Meal Tracking
- **Categorized Logging**: Quick-log meals by type (Healthy/Harmful) and category (Breakfast, Lunch, Dinner, Snack).
- **Desktop-Optimized History**: Professional data-table view for examining longitudinal dietary records.
- **Contextual Notes**: Add behavioral context to every entry for deeper AI analysis.

### 🎯 Milestone Command Center
- **Mission-Based Goals**: Define quantitative targets (e.g., "7 Healthy Meals Weekly") with real-time progress bars.
- **Streak Visualizer**: Animated "Healthy Streak" counter with fire-bounce effects to maintain momentum.
- **XP & Leveling System**: Earn experience points as you maintain dietary discipline.

### 🛡️ Secure System Infrastructure
- **Custom Node.js Backend**: Robust server architecture built for performance and security.
- **SQLite Data Engine**: High-speed local database for encrypted health data storage.
- **Session Security**: Industry-standard JWT (JSON Web Token) authentication and bcrypt password hashing.

---

## 🛠️ Technical Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion (Animations), Lucide Icons.
- **Backend**: Node.js, Express.js.
- **Database**: SQLite3 with Async wrapper.
- **Design**: Custom Teal-based Medical Design System with Glassmorphism.

## ⚡ Quick Start

### 1. Initialize Backend
```bash
cd server
npm install
npm start
```

### 2. Initialize Frontend (New Terminal)
```bash
npm install
npm run dev
```
